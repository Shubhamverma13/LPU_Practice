Insertion sort
__________________________
for(int i = 1; i < n; i++){
        int temp = arr[i];
        int j = i - 1;        
        for(; j >= 0; j--){
            if(arr[j] > temp){
                arr[j+1] = arr[j];
            }
            else {
                break;
            }
        }
        arr[j+1] = temp;
    }
-------------------------------------------------------------

Why
______________________

Adabtable
Stable
small size of an array
-------------------------------------------------------------

Space Complexity - > O(1)
____________________________

Time Complexity -> O(n^2)
-------------------------------------------------------------

Best case -> O(n)

-----------------

Worst case -> O(n^2)
